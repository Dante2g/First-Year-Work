Programming

In this unit we cover programming in a high level language called Java, we also practice problem solving, solution implementation, program design and testing. So far we have used Processing to learn Java and begin to create some basic games such as Tron Light Cycle and Pong.

Computer Systems Fundementals

CSF gives an introduction to mathematics and fundermentals underlying a computer system. So far we have learnt: Logic gates, Boolean algebra, how to build logic cicruits and how to construct karnaugh maps.
